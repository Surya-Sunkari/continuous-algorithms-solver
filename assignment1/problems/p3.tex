Prove the following statements about regularity in the $\ell_\infty$ norm.

\begin{enumerate}[(i)]
\item If $f : \mathbb{R}^d \to \mathbb{R}$ is twice-differentiable and $1$-smooth in $\ell_\infty$, $\operatorname{Tr}(\nabla^2 f(\mathbf{x})) \leq 1$ for all $\mathbf{x} \in \mathbb{R}^d$.\footnote{It may be helpful to use the characterization $\nabla^2 f(\mathbf{x})[\mathbf{v}, \mathbf{v}] \leq \|\mathbf{v}\|_\infty^2$ for all $\mathbf{v} \in \mathbb{R}^d$ (Lemma 14, Part II).}

\item If $f : \mathbb{R}^d \to \mathbb{R}$ is $1$-strongly convex in $\ell_\infty$, then
\[
\max_{\mathbf{x} \in \mathcal{X}} f(\mathbf{x}) - \min_{\mathbf{x} \in \mathcal{X}} f(\mathbf{x}) \geq \frac{d}{2}, \quad \text{for } \mathcal{X} := [-1, 1]^d.
\]
\end{enumerate}
